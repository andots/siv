# siv - Simple Image Viewer

siv is a simple image viewer made with Tauri and SolidJS.

## Features

- [x] Zoom in/out with mouse wheel.
- [x] Drop an image file on window.
- [ ] Traversing images in folder.

## Development

```bash
# dev
pnpm tauri dev
# build
pnpm tauri build
# release
pnpm release
```
